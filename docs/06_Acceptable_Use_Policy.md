# Acceptable Use Policy (AUP)
## Organization: [Company Name]
**Document Owner:** CISO / HR
**Version:** 1.3

### 1. Purpose
To outline the acceptable use of computer equipment, network resources, and internet access provided by the organization.

### 2. User Responsibilities
- Users must lock their workstations when unattended.
- Corporate hardware is strictly for business use, with incidental personal use permitted if it does not interfere with productivity or pose a security risk.

### 3. Unacceptable Activities
- Accessing, storing, or distributing illegal or offensive material.
- Disabling endpoint security agents (e.g., antivirus, EDR).
- Using corporate credentials to register for non-business-related third-party services.
- Attempting to bypass web filtering or network access controls.

### 4. Monitoring
The organization reserves the right to monitor, log, and audit all activities on corporate-owned devices and networks without prior notice to the user, in accordance with local laws.
